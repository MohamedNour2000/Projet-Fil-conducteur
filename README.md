Analyse de la Performance Économique (INPI 2024)
Dashboard Power BI Interactif : Du Design Figma à l'Intelligence Business.

🎯 Objectif du Projet
Ce projet vise à analyser le dynamisme économique français à travers les données réelles de l'INPI. L'enjeu était de transformer une base de données complexe en un outil d'aide à la décision stratégique, permettant d'identifier les secteurs les plus productifs et les zones géographiques à fort potentiel.

🛠️ Stack Technique
Design : Figma (Conception de l'UI/UX et du prototype).

Visualisation : Power BI Desktop.

Langages : DAX (Mesures complexes et insights dynamiques).

Traitement de données : Power Query (Nettoyage et modélisation).

Source : Données Open Data INPI 2024.

📈 Fonctionnalités Clés
Le dashboard est articulé autour de 3 pages stratégiques :

1. Vue d'Ensemble (Synthèse Nationale)
Suivi du Chiffre d'Affaires global et des tendances de croissance.

Analyse de la répartition sectorielle.

2. Intelligence Géographique (Analyse Locale)
Cartographie interactive des flux financiers par commune (Niort, Dinan, Agen, etc.).

Top 10 des zones les plus contributrices au CA national.

3. Matrice de Performance & Productivité
Analyse de Productivité : Calcul réel du CA par salarié pour identifier l'efficacité opérationnelle par secteur.

Segmentation : Répartition des entreprises par taille (Petites, Moyennes, Grandes) avec un tri logique personnalisé.

Insight Dynamique : Génération automatique de commentaires textuels basés sur les filtres actifs.

💡 Concepts DAX Avancés Utilisés
Pour ce projet, j'ai mis en œuvre des solutions techniques pour garantir l'intégrité des données :

Productivité Sectorielle : Utilisation de la fonction DIVIDE pour sécuriser les calculs de ratios.

Storytelling Automatisé : Création de mesures de texte dynamique avec TOPN et CONCATENATEX pour interpréter les données en temps réel.

Tri Personnalisé : Modélisation d'une table d'index pour forcer l'ordre logique des tranches d'effectifs (TPE > PME > GE).

🚀 Comment utiliser ce projet ?
Télécharger le fichier .pbix présent dans ce repository.

Ouvrir avec Power BI Desktop.

Utiliser la barre de navigation latérale (boutons invisibles superposés au design) pour naviguer entre les analyses.
